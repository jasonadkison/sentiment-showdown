# sentiment-showdown

## Requirements
ruby 2.3.1p112

## What it does
Calculates sentiment from sample data and displays the results.

## How to run it
1. git clone git@github.com:mattr-co/sentiment-showdown.git
1. `cd sentiment-showdown`
1. `bundle install`
1. `ruby main.rb`

## Where to put sample data
`data/sentiment.yml`
